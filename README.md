Web server template for building a React app with an Express.js backend.

Packages included:
 * express
 * react
 * react-dom
 * dotenv
 * morgan
 * prettier
 * typescript

Running the dev server:
```
npm run dev
```

Linting your code:
```
npm run lint
```

Autoformatting your code using prettier:
```
npm run format
```
